score(idx)¶

Calculates silhouette coefficients of clustering at a given epoch

a_i = cohesion (mean within-cluster distance) b_i = separation (minimum mean between-cluster distance) s_i = silhouette coefficient

s_i = (b_i - a_i) / max(a_i, b_i)

Parameters

idx – epoch to calculate silhouette coefficients

Returns

array of silhouette coefficients
